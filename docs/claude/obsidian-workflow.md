# Obsidian Workflow

## Local REST API Integration
- Obsidian Local REST API runs on port `27123`.
- API key is stored in `.claude/settings.json`.
- Used for reading and writing notes from Claude Code CLI.

## Usage Notes
- Notes can be created, read, and updated via the REST API during development sessions.
- Use this for capturing decisions, TODOs, and session logs while working on the project
